## 게임 난이도 추가

- [x] 난이도별 7단계로 확정 (10단계 계획에서 변경됨). 어려움 주제 6→7개로 1개 추가("이름이 영어로 적힌 물건"), 쉬움 10개 중 7개만 사용, 보통 8개 중 7개만 사용 (매일 시드 셔플로 7개 선택)
- [x] 난이도별 진행상황을 localStorage에 독립적으로 저장 (`chalkak_easy`/`chalkak_medium`/`chalkak_hard`)
- [x] 실패 제한: 쉬움 무제한 / 보통 단계마다 3번 실패 시 종료(다음 단계 진입 시 리셋) / 어려움 한 판 누적 5번 실패 시 종료. 게임오버 시 "N단계에서 실패" 표시 + 난이도 재선택 버튼
- [ ] 보통/어려움: 하루 한 번만 플레이 가능 (완료 후 재도전 막기) — 아직 미구현, 지금은 난이도 무관하게 자유롭게 재도전 가능
- [ ] 강제종료 후 재진입 시 진행하던 단계부터 이어서 — localStorage 저장은 이미 되고 있어 기본 동작은 되지만, "완료 후 재도전 방지"와 함께 별도로 검증 필요
- [ ] 주제 0시 초기화: 현재 기기 로컬시간 기준 (todayKey()). 랭킹모드 연동 시 서버 시간 기준으로 바꿀지 추후 결정
