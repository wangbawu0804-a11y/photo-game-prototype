# 기획서

## 주제
오늘의 찰칵 — 매일 제시되는 주제에 맞는 사진을 찍어 올리면 AI(Claude API)가 사진이 주제에 부합하는지 판정해주는 포토 챌린지 게임.

## 형태 결정
**웹 (Web)**
이유: 별도 설치 없이 링크만으로 바로 즐기는 캐주얼 게임 특성상 접근성이 중요하고, `index.html` 단일 페이지 + Vercel Serverless Function 조합으로 이미 구현이 시작되어 있음.

## 현재 파이프라인 단계
서버(Server/Backend) 단계 진행 중
- 화면(UI): `index.html`에 완료 (사진 프레임, 카운터, 주제 카드 등 구현됨)
- 서버: `api/judge.js` (Vercel Serverless Function, Claude API로 사진 판정) 구현됨
- DB: 아직 없음
- 배포: 아직 확인 안 됨 (로컬 동작 확인 필요)
